1、关闭文档 CTRL + W

2、存储 CTRL + S （替存）

3、存储为 CTRL + SHIFT + S 另存一份文件

4、格式：

  • .psd PS 源文件格式，图层、文字、样式等，可再次编辑（给自己）

  • .jpg 有损压缩格式，不能更改（给客户），品质最高12

    • 5、文件打开
  • (1）文件下拉菜单-打开（CTRL+o）
  • (2) 双击空白软件处也能打开

    • 6、CTRL + T 可以改变图像的大小

    7、图层面板快捷键 F7 ,其实图层就是一张张透明的纸，可以实现叠加问题，在图层面板中，排序在上面的图层移动的时候可以压住下面的图层

    8、复制 按住 ALT 键

    9、合并图层：ctrl + e 可以合并图层

    10、魔棒工具快速选择选区，容差越大选的越细，此时选择背景，然后反选 CTRL + SHIFT + I 可以选取目标内容

    11、每做一个步骤不要忘记新建一个图层，否则的话新加的东西就不能移动

    12、有些细节选不上的话，可以摁着 SHIFT 键加选区

    13、钢笔工具：P
  • 作用：绘制路径、生成选区，抠图。
  • 组成：路径线和锚点
  • 绘制路径后，CTRL + 回车，生成选区。
  • 路径类型 ：
  • 直线性路径：连续单击
  • 曲线形路径：第一点单击，第二点拖动鼠标。
    • 14、切片：
  • 利用切片工具手动划出
  • 图层···新建基于图层的切片
  • 利用标尺，基于参考线的切片（选择切片工具）
  • 先选一个整个的切片，切片选择工具--属性面板中有划分--可以等份数划分
  • 导出切片：文件存储为 web 设备所用格式，保存时切片选项选择：选中的切片
