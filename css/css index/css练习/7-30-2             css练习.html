中东部高温“燃”至月底 华南等局地强降雨

2018年7月27日 3:04 来源： 新华网

河南、湖北、重庆、江西、湖南等地正遭遇罕见连续高温过程，并且高温闷热天气还将持续至月底！高温当道，雨水打起“游击战”，今(27日)起三天，黄淮、江淮、东北等地多分散性强对流天气，华南、云南等部分地区仍多强降雨。

热！近日我国中东部地区出现高温热浪天气，河南、湖北、重庆、江西、湖南等地遭遇罕见连续高温过程，其中重庆更是连续7天出现40℃以上的极端高温天气，刷新当地连续40℃以上天数纪录，昨天虽未能继续刷新纪录，最高气温也达到37℃以上，酷热难耐。 从13日以来的这轮大范围高温过程，截至昨天已持续了13天，目前仍然看不到终结迹象。从连续性、夜温、闷热等因素看，此轮高温过程特征显著！

持续性方面，省会级城市里，重庆连续高温达到13天，武汉12天，郑州达到11天，西安、南昌分别达到9天和8天，其余城市基本都在“安比”影响下曾短暂退出高温，但之后又高温重燃；不仅如此，高温在一天中的持续时间也很长，如重庆最近一周每天35℃以上时间，至少占满8小时，甚至超12小时以上，高温全覆盖白天活动时段，而且因为湿度大，体感温度甚至可高出10℃！

预计今起至月底，四川盆地、江南、江淮等地还将出现较大范围的高温天气，其中高温中心将在重庆湖北湖南江西等地。中央气象台今晨继续发布高温黄色预警：预计今天白天，内蒙古西部、陕西关中地区、四川盆地东部、重庆、江汉、江淮、江南等地最高气温有35～36℃，其中，内蒙古西部、陕西关中地区、重庆、湖北南部、湖南大部、江西中北部、浙江中西部、福建中部等地的部分地区最高气温可达37～39℃，重庆北部局地可达40℃以上。
